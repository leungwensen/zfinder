# ZFinder

[ZFinder](https://github.com/leungwensen/zfinder) is designed to replace the finder.app in Mac (or file explorer in Windows). with the magic of modern Browsers, we can do a lot more.

## usage

```shell
npm install -g zfinder
zfinder -h
```
